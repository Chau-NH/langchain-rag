# LANGCHAIN - RAG

## Prerequisites

- NodeJS version >18.x

## Installation

```sh
npm install -S langchain
```

## Env variables

- `OPENAI_API_KEY` - Secret key of OpenAI
- `PORT` - port on which application is running, by default it's `3000` (in Dockerfile)
